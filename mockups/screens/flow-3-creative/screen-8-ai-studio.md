## Flow 3: Creative Studio

### Screen 8: AI Studio

**Prompt:**

Design the AI Generation Input screen (Coloring Pages).

**Layout:**

- **Header**: "AI STUDIO".
- **Prompt Input**: Large text area in the center.
- **Style Selector**: Horizontal scroll of chips (e.g., "Line Art", "Doodle", "Detailed").
- **Footer**: "Generate (1 Credit)" button.

**Visual Centerpiece:**
The Prompt Input field. It should feel like a command line or a creative writing canvas.

**Key Elements:**

- **Input**: "Describe what to draw..." placeholder. Minimal border.
- **Credit Balance**: "95 Credits Available" pill in top right.
- **History**: "Recent Creations" horizontal list below input.
- **Magic Button**: "Surprise Me" (Die icon) to fill random prompt.

**Typography:**

- Input text: Large (24px) serif or monospace. Creative feel.

**Interaction Notes:**

- "Generate" button is disabled if input is empty.
- Pressing Generate triggers a full-screen loading animation (see Screen 9).

**Vibe:**
Magic canvas. Infinite possibilities.
