## Flow 3: Creative Studio

### Screen 10: Recipe Import

**Prompt:**

Design the Recipe Import screen.

**Layout:**

- **Input Group**: "Paste URL" field + "Import" button.
- **Manual Entry**: "Or write manually" link.
- **Preview Card**: The resulting recipe card template.

**Visual Centerpiece:**
The transformation from "Messy URL" to "Clean Recipe Card".

**Key Elements:**

- **URL Input**: "https://..."
- **Loading**: "Scraping ingredients..." / "Parsing instructions..." steps shown.
- **Result Editor**:
  - Title (Editable).
  - Ingredients List (Editable).
  - Steps (Editable).
  - "Servings" / "Time" inputs.

**Typography:**

- "RECIPE" header in serif font (e.g., 'Instrument Serif') to distinguish from the industrial parts of the app. Cooking is organic.

**Interaction Notes:**

- Import button triggers a spinner.
- Success shows the filled card below.

**Vibe:**
Sous-chef. Helpful and organized.
