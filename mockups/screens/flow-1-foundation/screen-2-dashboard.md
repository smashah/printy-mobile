## Flow 1: Foundation

### Screen 2: Dashboard (Template Hub)

**Prompt:**

Design the Main Dashboard (Home) for "Printy".

**Layout:**

- **Top Bar**: "Templates" header (Left), Connection Status Indicator (Right).
- **Content**: Masonry or Grid layout of "Blueprints".
- **Bottom Bar**: Navigation (Home, Studio, Pro, Settings).

**Visual Centerpiece:**
The "Blueprints" (Templates). Each represents a printable item (Wifi Card, Todo List).

**Key Elements:**

- **Connection Indicator**: A pill badge. Red "Disconnected" or Green "Connected: PM-241-BT". Tapping it goes to Device Scan.
- **Category Tabs**: "All", "Utility", "Fun", "Dev" (Segmented Control style, 2px borders).
- **Template Card**:
  - White background, 2px black border, hard shadow.
  - Preview image (dithered) of the template.
  - Title (e.g., "WIFI CARD") in monospace.
  - Tag (e.g., "FREE" or "PRO").
- **Fab (Floating Action Button)**: Optional, maybe for "Quick Print".

**Typography:**

- Section Headers: "FRESH INK" (New templates), "ESSENTIALS" (Core).

**Interaction Notes:**

- Tapping a card opens the Template Editor.
- Tapping Connection Indicator opens Device Scan.

**Vibe:**
A well-organized workshop toolboard. Ready to use.
